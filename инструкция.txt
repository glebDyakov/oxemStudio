Вызывайте в консоли javaScript функции в ответ на вызовкоторых будет возвращаться JSON и происзходить запросы в базу данных.
У меня присутствует 2 query-параметра page и order.
query-параметр order отвечает за сортировку товаров в таблице и принимав себя значение asc для сортировки по возрастанию или значение desc для сортировке по убыванию.
query-параметрpage отвечает за текущую страницу пагинации.
